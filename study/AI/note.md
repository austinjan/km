1. 了解 LLM 本質
   1. LLM 是一種統計模型，透過大量文本資料訓練而成，能夠預測下一個字詞的機率。
   2. LLM 並不具備真正的理解能力，而是基於訓練資料中的模式來生成回應。
2. 設計有效的 Prompt
   1. 明確且具體：提供清晰的指示，避免模糊不清的要求。
   2. 提供上下文：給予足夠的背景資訊，讓模型能更好地理解需求。
   3. 使用範例：提供範例輸入和輸出，有助於模型理解預期結果。
3. Context engineering 技巧
   1. 分段提示：將複雜的任務拆分成多個簡單的步驟，逐步引導模型完成。
   2. 反覆迭代：根據模型的回應，不斷調整和優化 Prompt，以達到更好的效果。
   3. 使用系統提示：利用系統級別的提示來設定模型的行為和風格。
   4. Progressive disclosure：逐步揭示資訊，避免一次性提供過多內容，讓模型能更專注於當前任務。
