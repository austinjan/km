# Tool Name
- write-scratchpad
- read-scratchpad
- list-scratchpads-sections

## Brief

## Description
A working memory tool for organizing thoughts, tracking progress, and maintaining context during complex tasks.

The scratchpad persists throughout the session and can be read back at any time.

## When to Use
- Record key findings while analyzing large files.
- Track progress in multi-step tasks.
- Maintain a structured summary to avoid losing focus.
- Store intermediate results for later reference

## Usage Notes
- Use sections to organize different types of Notes
- Update existing sections rather than creating duplicates
- Keep notes concise - this is working memory, not documentation
- Read the scratchpad periodically to stay on track
- Clear completed items to reduce noise

## Predefined Sections
| Section | Purpose |
| --- | --- |
| objective | Current goal and success criteria |
| finding | Key discoveries and facts |
| todo | Remaining tasks |
| questions | Uncertainties to resolve, issues have to disscuss with user |
| scratch | Temporary calculations or drafts |
